# DevFlow AI 🚀

AI-powered DevOps Assistant built with React, Express, and Gemini AI.

## Features
- AI CI/CD Pipeline Generator
- AI Dockerfile Generator
- AI Error Analyzer
- Modern DevOps Dashboard

## Tech Stack
- React + Vite
- TailwindCSS
- Express.js
- Gemini AI

## Run Project

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
